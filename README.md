# 🛍️ Shoplane – Full Stack E-commerce Application

A full-stack e-commerce web application built using **Java Spring Boot** for the backend and **HTML, CSS, JavaScript** for the frontend.
The application simulates a real-world online shopping experience with product browsing, cart management, and order processing.

---

## 🚀 Features

* 🛒 Product listing with dynamic data from backend
* 🔍 Live product search functionality
* 🧺 Add to cart with real-time updates
* 💳 Multi-step checkout flow (Cart → Details → Order Confirmation)
* 🔗 RESTful API integration between frontend and backend
* ⚙️ Backend built using Spring Boot with JPA
* 📱 Fully responsive UI (mobile + desktop)

## ⚙️ How to Run the Project

### 🔹 1. Clone the Repository

```
git clone https://github.com/your-username/Shoplane.git
cd Shoplane
```

---

### 🔹 2. Run Backend (Spring Boot)

```
cd backend
mvn spring-boot:run
```

👉 Backend will start at:
`http://localhost:8080`

---

### 🔹 3. Run Frontend

* Open `frontend/index.html` in browser
  OR
* Use Live Server (VS Code recommended)

## 🔗 API Endpoints (Sample)

* `GET /products` → Fetch all products
* `GET /products/{id}` → Get product details
* `POST /orders` → Place order
